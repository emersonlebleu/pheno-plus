# Pheno-Plus
Pheno-Plus is a SMART on FHIR application designed to streamline phenotype extraction from electronic health records (EHR). It enables users to select relevant clinical notes, extract phenotype terms using ClinPhen, review terms in context, and curate a final phenotype list in multiple formats.

# üöÄ Features
- EHR Integration: Embeds directly within the clinical chart using SMART on FHIR.
- Flexible Note Selection: Users can choose which clinical notes to analyze.
- Automated Phenotype Extraction: Utilizes ClinPhen NLP to extract relevant phenotype terms.
- Contextual Review: Displays terms within the text for user validation.
- Interactive Curation: Users can unselect non-relevant terms before finalizing.
- Easy Data Export: Copy the final phenotype list in multiple formats.
